SimpleSwing Addon

Quick Install:
1. Extract the SimpleSwing folder into:
   Interface/AddOns/
2. Launch the game.

Commands:
/ss  - Opens SimpleSwing menu

WeakAura (for Slam sound):
1. Type /wa
2. Click Import
3. Paste the string from WeakAuraImport.txt
4. Done

Notes:
- Includes original Slam1.wav sound.
- Sound triggers on Slam using WeakAura.
- Currently set up for MH/2H use.
